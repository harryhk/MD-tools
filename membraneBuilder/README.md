# membraneBuilder  
It is a collection of python tools that can be used to build membrane bilayers or mixed membrane blayers. 

## include 
The following is only a general description of each tools listed in this folder. Each tools has more functionality supported. For further information, please run the code with -h to see the helpdoc. 

* membrane_builder.py:
	
	* set up a two dimensional grid and place lipids and cholesterols on to the grid. 
	* has many useful classes such as Molecule to create molecules from gro file and perform rotation and translocation one them.

* remove_water.py:
	
	* delete water between certain range [ zmax, zmin ] and renumber the molecule and atoms.
